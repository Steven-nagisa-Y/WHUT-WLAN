## 使用方法
1. 修改./user.json为你的校园卡号和密码
   
2. 安装依赖（目前只有 requests

    ```shell
    pip3 install requests
    ```

    ```shell
    python3 main.py
    ```

RasberryPi 文件夹下面是在树莓派下运行的 python 代码以及启动脚本，自动登陆并将 ip 发送到指定邮箱
